skein keystore get¶

Get a value from the keystore

usage: skein keystore get [--key KEY] [--wait] [--help] APP_ID
app_id¶

The application id. To use in a container during a skein job, pass in “current”

--key <key>¶

The key to get. Omit to the whole keystore.

--wait¶

If true, will block until the key is set

--help, -h¶

Show this help message then exit
